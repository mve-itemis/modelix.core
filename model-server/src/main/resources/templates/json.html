JSON API

    GET /{repositoryId}/ Returns the model content of the latest version on the master branch.
    GET /{repositoryId}/{versionHash}/ Returns the model content of the specified version on the master branch.
    GET /{repositoryId}/{versionHash}/poll
    POST /{repositoryId}/init Initializes a new repository.
    POST /{repositoryId}/{versionHash}/update Applies the delta to the specified version of the model and merges it into the master branch. Return the model content after the merge.
    WEBSOCKET /{repositoryId}/ws WebSocket for exchanging model deltas.
